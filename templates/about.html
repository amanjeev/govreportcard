What this website is

This website is just an effort start a conversation on the Indian government's agenda and items in manifesto. The main goal of this project is to establish a platform where the Indian citizens can convery their thoughts on each of the items in the current serving majority's manifesto.

This website hopes to serve as a record of what the citizens think about a particular item of the manifesto, discuss it and hope to see if there has been any progress in it.

It is a geniune attempt at a project that might grow into tracking citizens' input and thoughts, hopefully creating a better democracy.

What this website isn't

This website is in no way a place to fuel hatred against any particular gender, caste, religion, status, position, nationality, party, opinions, thoughts.

This website is not biased in any way.
